# Chingu (친구)

The open source Chrome extension for SOPs that actually work.
Free forever. Built in public.

## Why

Every SOP tool today either breaks during capture, paywalls the basics,
or costs $20/seat/month. Ops managers at 5-50 person teams deserve better.

Chingu is a free, open source Chrome extension that captures browser
workflows and turns them into clean, shareable step-by-step guides.
The screenshots actually work. The branding is free. The code is yours.
